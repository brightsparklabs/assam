# Assam

Interfaces for modelling Abstract Syntax Notation (ASN.1).

## Setup and Build

```bash
# bash
git clone git@github.com:brightsparklabs/assam.git
cd assam
./gradlew build
# generate intellij classpath (optional)
./gradlew idea
```
